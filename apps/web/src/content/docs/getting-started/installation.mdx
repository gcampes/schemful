## Requirements

- **Node.js 18** or later
- A Contentful space with a management token

## Install CTKit

You can install CTKit as a project dependency (recommended) or globally.

### Project dependency

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install @ctkit/cli
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @ctkit/cli
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @ctkit/cli
    ```
  </TabItem>
</Tabs>

Then run it with `npx ctkit`, `pnpm ctkit`, or `yarn ctkit`.

### Global install

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install -g @ctkit/cli
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add -g @ctkit/cli
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn global add @ctkit/cli
    ```
  </TabItem>
</Tabs>

## Verify installation

```bash
ctkit --version
```

You should see the installed version printed to stdout. If you installed locally, use `npx ctkit --version`.
